# Medication Reconciliation

Generate a medication reconciliation report for the patient. Include:

## Current Medications
List all active medications with:
- Drug name and dosage
- Start date
- Indication (reason prescribed)

## Discontinued Medications
List recently stopped medications with:
- Drug name
- Start and stop dates
- Reason for discontinuation (if known)

## Flags
- Duplicate therapies (same drug class prescribed multiple times)
- Missing indications (medications without a clear linked condition)
- Potential interactions (if any)

## Summary
Brief clinical summary of the medication profile.
