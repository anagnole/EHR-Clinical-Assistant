# EHR Graph Schema

Define the EHR graph schema with 7 node tables (Patient, Encounter, Condition, Medication, Observation, Procedure, Provider) and 12 relationship tables (HAS_ENCOUNTER, HAS_CONDITION, HAS_MEDICATION, HAS_OBSERVATION, HAS_PROCEDURE, ENCOUNTER_DIAGNOSIS, ENCOUNTER_MEDICATION, ENCOUNTER_OBSERVATION, ENCOUNTER_PROCEDURE, TREATED_BY, PRESCRIBED_BY, ORDERED_BY). Register this as the instance's custom schema.
